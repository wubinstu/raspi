### 运行流程

```text
处理运行参数 -> (检查PID文件) -> (进入守护进程模式) ->
初始化树莓派 -> 设置还原点 -> 注册信号 ->
读取配置文件 -> (读取证书密钥) -> 建立TCP链接 ->
(SSL握手) -> 开启数据采集线程 -> 发送数据

以上(括号)表示非必须步骤
```
